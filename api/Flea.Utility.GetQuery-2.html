Include(String[])

Include one or more sub-entities in the query. This is used when you want to fetch an entity whose model contains foreign keys and you want to fetch the foreign objects as well. In the following example it is used to fetch a Reservation in addition to its properties referencing Customer and Event objects.

	await factory
		.Query<BingoContext, Reservation>()
		.Include("ReservationOwner", "Event")
		.First(r => r.Id == 4);
